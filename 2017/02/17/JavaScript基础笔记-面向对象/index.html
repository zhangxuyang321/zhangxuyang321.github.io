面向对象

robot对象有名字,有身高,很像小明,所以就根据它来创建小明

创建对象

JavaScript对每个创建的对象都会设置一个原型,指向他的原型对象.当我们用obj.xxx访问一个对象的属性时,JS引擎会现在当前对象上查找该属性,如果没找到就到其原型对象上找,如果还没找到就一直上溯到Object.prototype对象,最后如果还是没有找到,就返回undefined.
